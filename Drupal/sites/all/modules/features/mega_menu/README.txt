To use this feature, Install/enable dependencies:

  context
  context_ui
  features
  libraries
  menu
  special_menu_items
  superfish

Perform the following steps:

  1. Enable Superfish for the given menu block.
  admin/structure/block/manage/menu/<delta>/configure

  2. Style should be set to "none" on superfish block.

  3. To create columns, floow this menulink structure:

  Parent 1
  __Column 1
  ____Link 1
  ____Link 2
  __Column 2
  ____Link 1
  ____Link 2

  Parent 2

  Parent 3
